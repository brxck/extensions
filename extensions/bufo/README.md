# Bufo

Search and use the full [bufo.fun](https://bufo.fun) catalog of bufo stickers right from Raycast. Browse over 1,200 bufos in a scrollable grid, then paste, copy, or download any one of them in a keystroke.

## Features

- Search the catalog by name and tag to find exactly the bufo you need
- Browse thumbnails in a fast, scrollable grid
- Paste a bufo directly into the frontmost app with a single keypress
- Copy the image file to your clipboard (including animated GIFs)
- Download any bufo to a folder on disk

## Usage

Open the **Search Bufos** command in Raycast and start typing to filter the catalog. Press **Enter** to paste the selected bufo into whatever app you were using. Use **⌘C** to copy the image to your clipboard, or **⌘⇧D** to download it to disk.

Two preferences let you tailor the experience:

- **Default Action** — choose whether Enter pastes, copies, or downloads (defaults to Paste)
- **Download Directory** — set the folder where downloaded bufos are saved (defaults to `~/Downloads`)
